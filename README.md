Hello! I will help you step-by-step into setting this simple OS.

First, you want to install Python on your Windows 11, make sure that these imports are installed, too:
tkinter
os
sys
If you don't know how to install these imports, just simply install Python, and in the Command Shell (Of Windows, not Python), type this:
pip install tkinter
pip install os
pip install sys
These should already be installed, but just to be safe, I left them there.

Next, press "Win + R" to open the Run Screen. Type in this:
shell:startup
It should open a folder. Put the "Py-OS Main.py" in there.

Finally, restart the system and sign in. It should open up the Python code! If it doesn't then you either did something wrong, you edited the file, or you just have to give it a second.
